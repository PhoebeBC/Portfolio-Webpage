  • I developed a script capable of encrypting a message from a text file using a Caesar cipher and generating another text file with the encrypted message.
  • Subsequently, I enhanced the complexity of the encryption process. Rather than all characters sharing the same shift, each succeeding character experienced a shift of n+1, where n represents the shift of the preceding character.
